# movilidad_back
Servicios de movilidad

## Conductores (api/drivers)

Endpoint de conductores que contiene la funcionalidad de listar los conductores.
Está protegido con seguridad por JWT.

## Carros (api/cars)

Endpoint de carros que contiene las funcionalidades de consultar, crear, editar y eliminar un carro.
Está protegido con seguridad por JWT.

## Desplazamientos (api/displacements)

Endpoint que contiene las funcionalidades de consultar, crear, editar y eliminar un desplazamiento.
Está protegido con seguridad por JWT.
